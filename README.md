# track-ur-shows
A full-stack web application, focused on movies and TV series tracking, built with modern web technologies.

Frontend: React, Next.js (Typescript)
Backend: Next.js API routes, Node.js
Database: PostgreSQL

This application allows users to:
  - Browse movies and TV series (via external API integration)
  - Create and manage personal watchlist
  - Track watching status (watching, completed, plan to watch)
  - AI powered assistant for movie/series related queries

External Integrations:
  - Movie/TV series data API: TMDB API
